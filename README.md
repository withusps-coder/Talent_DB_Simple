# 나만의 인재 DB 검색기

간단한 CRUD 기능을 테스트하는 프로젝트입니다.  
**CLI 버전**과 **웹 버전** 2가지 방식으로 사용할 수 있습니다.

## 🎯 핵심 기능

✅ **입력**: 후보자 이름, 연락처, 핵심 스킬, 연차 입력  
✅ **저장**: JSON 파일로 데이터 저장  
✅ **조회**: 전체 목록 조회 및 검색  
✅ **삭제**: 후보자 삭제 (웹 버전 전용)  
✅ **웹 UI**: 깔끔한 웹 인터페이스 제공  

## 🚀 사용 방법

### 방법 1: CLI 버전 (터미널)

#### 실행
```bash
python3 talent_db.py
```

#### 메뉴
1. 📝 후보자 등록
2. 📋 전체 목록 조회
3. 🔍 후보자 검색
4. 🚪 종료

---

### 방법 2: 웹 버전 (브라우저) ⭐ 추천!

#### 1. Flask 설치
```bash
pip install flask
```

#### 2. 서버 실행
```bash
python3 app.py
```

#### 3. 브라우저 접속
서버가 시작되면 브라우저에서 다음 주소로 접속하세요:
```
http://localhost:5000
```

#### 4. 웹 UI 사용
- ✅ **후보자 등록**: 폼에 정보 입력 후 "등록" 버튼 클릭
- ✅ **전체 목록**: 자동으로 목록 표시
- ✅ **검색**: 검색창에 이름 또는 스킬 입력
- ✅ **삭제**: 각 후보자 카드의 "🗑️ 삭제" 버튼 클릭

#### 5. 서버 종료
터미널에서 `Ctrl + C`

## 📦 저장 방식

- **파일**: `candidates.json`
- **형식**: JSON
- **구조**:
```json
[
  {
    "id": "20260122091000",
    "name": "홍길동",
    "contact": "010-1234-5678",
    "skills": "Python, JavaScript",
    "experience": "3년",
    "created_at": "2026-01-22 09:10:00"
  }
]
```

> **참고**: CLI 버전과 웹 버전은 동일한 `candidates.json` 파일을 공유합니다.

## 💡 테스트 방법

### 1. 후보자 등록
```
이름: 홍길동

연차: 3년
```

### 2. 목록 조회
등록된 모든 후보자를 확인할 수 있습니다.

### 3. 검색
이름 또는 스킬로 검색 가능합니다.

## 🔧 기술 스택

### CLI 버전
- **Python 3.9+**
- **JSON**: 데이터 저장
- **표준 라이브러리만 사용** (추가 설치 불필요)

### 웹 버전
- **Flask**: 백엔드 서버
- **HTML/CSS/JavaScript**: 프론트엔드
- **REST API**: JSON 기반 API
- **반응형 디자인**: 모바일/데스크톱 대응

## 📁 파일 구조

```
Talent_DB_Simple/
├── talent_db.py           # CLI 버전 (터미널용)
├── app.py                 # 웹 버전 (Flask 서버)
├── candidates.json        # 데이터 파일 (자동 생성, 공유)
├── templates/
│   └── index.html         # 웹 UI HTML
├── static/
│   ├── style.css          # 웹 UI 스타일
│   └── script.js          # 웹 UI 스크립트
└── README.md              # 이 파일
```

## 🎨 웹 UI 특징

✨ **모던 디자인**: 깔끔하고 직관적인 인터페이스  
✨ **실시간 검색**: 타이핑하면 즉시 결과 표시  
✨ **카드 레이아웃**: 후보자 정보를 카드 형태로 표시  
✨ **반응형**: 모바일/태블릿/데스크톱 모두 지원  
✨ **애니메이션**: 부드러운 전환 효과  

## 🎓 이 프로젝트로 배울 수 있는 것

### CLI 버전
✅ **CRUD 기본**: Create, Read 구현  
✅ **파일 I/O**: JSON 읽기/쓰기  
✅ **CLI 프로그램**: 사용자 입력 처리  
✅ **데이터 구조**: 리스트와 딕셔너리 활용  

### 웹 버전
✅ **Flask 웹 서버**: 백엔드 서버 구축  
✅ **REST API**: JSON 기반 API 설계  
✅ **프론트엔드**: HTML/CSS/JavaScript 연동  
✅ **비동기 통신**: Fetch API 활용  
✅ **CRUD 완전 구현**: Create, Read, Update, Delete  

**이 구조를 이해하면 풀스택 웹 애플리케이션을 만들 수 있습니다!** 🚀

## 🆚 CLI vs 웹 버전 비교

| 기능 | CLI 버전 | 웹 버전 |
|------|----------|---------|
| 설치 | 추가 설치 불필요 | Flask 설치 필요 |
| 사용성 | 터미널 사용 | 브라우저 사용 (편리) |
| UI | 텍스트 기반 | 그래픽 UI |
| 검색 | 수동 입력 | 실시간 검색 |
| 삭제 | ❌ 미지원 | ✅ 지원 |
| 접근성 | 로컬만 | 네트워크 공유 가능 |
