# 나만의 인재 DB 검색기

간단한 CRUD 기능을 테스트하는 프로젝트입니다.

## 🎯 핵심 기능

✅ **입력**: 후보자 이름, 연락처, 핵심 스킬, 연차 입력  
✅ **저장**: JSON 파일로 데이터 저장  
✅ **조회**: 전체 목록 조회 및 검색  

## 🚀 사용 방법

### 실행
```bash
python3 talent_db.py
```

### 메뉴
1. 📝 후보자 등록
2. 📋 전체 목록 조회
3. 🔍 후보자 검색
4. 🚪 종료

## 📦 저장 방식

- **파일**: `candidates.json`
- **형식**: JSON
- **구조**:
```json
[
  {
    "id": "20260122091000",
    "name": "홍길동",
    "contact": "010-1234-5678",
    "skills": "Python, JavaScript",
    "experience": "3년",
    "created_at": "2026-01-22 09:10:00"
  }
]
```

## 💡 테스트 방법

### 1. 후보자 등록
```
이름: 홍길동
연락처: 010-1234-5678
핵심 스킬: Python, JavaScript, React
연차: 3년
```

### 2. 목록 조회
등록된 모든 후보자를 확인할 수 있습니다.

### 3. 검색
이름 또는 스킬로 검색 가능합니다.

## 🔧 기술 스택

- **Python 3.9+**
- **JSON**: 데이터 저장
- **표준 라이브러리만 사용** (추가 설치 불필요)
